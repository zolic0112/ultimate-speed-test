## 📞 故障排除快速参考

### 问题：GLTFLoader 脚本错误
**症状：** 控制台显示 "THREE.GLTFLoader not available"
```
解决方案：
1. 检查 index.html
2. 确认 GLTFLoader 脚本已添加
3. 脚本应该在 Three.js 之后
4. 如果使用 CDN，检查 URL 是否正确
5. 重新加载页面
```

### 问题：404 - 找不到 .glb 文件
**症状：** 控制台显示 "404 Not Found"，奖牌不显示
```
解决方案：
1. 检查文件是否存在于 app/medals/
2. 检查文件名是否完全正确
3. 检查 medal.js 中的路径是否匹配
4. 确保使用相对路径（medals/medal-s.glb）
5. 不要使用绝对路径或 C:\ 路径
```

### 问题：模型加载很慢
**症状：** 模型需要 > 3 秒才能显示
```
解决方案：
1. 检查 .glb 文件大小
2. 如果 > 200 KB，在 Blender 中启用 Draco 压缩
3. 考虑简化网格（减少三角面数）
4. 优化纹理分辨率
5. 使用浏览器缓存（生产环境）
```

### 问题：模型方向或大小错误
**症状：** 奖牌看起来被旋转或缩放不正确
```
解决方案：
1. 在 medal.js 的 _swapModel() 中调整：
   - 缩放：this.body.scale.multiplyScalar(1.2)
   - 旋转：this.body.rotation.x = Math.PI / 2
2. 或者回到 Blender，修复模型，重新导出
```

### 问题：纹理/材质看起来不对
**症状：** 颜色或反光不如预期
```
解决方案：
1. 确保在 Blender 中使用了 Principled BSDF
2. 检查 Blender 导出选项：
   - 确认 "Include Materials" 已选中
   - 确认 "Include Textures" 已选中
3. 在 Three.js 编辑器中预览 .glb 文件
4. 如果看起来不对，回到 Blender 调整材质
```
